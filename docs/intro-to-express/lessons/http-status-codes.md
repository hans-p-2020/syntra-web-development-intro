# HTTP Status Codes

## Successful Requests

The web services return a HTTP 2xx response code when the request was successful.

**HTTP Success Codes**:

| Success Code | Message | Description                            |
| ------------ | ------- | -------------------------------------- |
| 200          | OK      | The request was received successfully. |
